⚙️ Firebase Setup Required (one-time, ~5 min)

Replace the firebaseConfig placeholder in this file's <script> to enable real-time sync across everyone's phones.

  1. Go to console.firebase.google.com → Create project
  2. Project Settings → Add Web App → copy firebaseConfig
  3. Build → Realtime Database → Create database → Test mode
  4. Paste your config into the <script> at the bottom of this file
  5. Push to your GitHub Pages repo and share the link!
